Weather App is a React-based web application that provides real-time weather updates and a 5-day forecast using the OpenWeather API. The application features city-based search, dynamic weather icons, loading states, error handling, and a responsive glassmorphism-inspired user interface.

Features:
- Current weather information
- 5-day weather forecast
- Search by city name
- Weather icons
- Loading and error states
- Keyboard support (Enter key search)
- Responsive design
- Environment variable support for API keys
